# 项目模型清单

## 1. DeepSeek Chat (deepseek-chat)

| 项目           | 内容                                                              |
| -------------- | ----------------------------------------------------------------- |
| **介绍**       | DeepSeek V3 大语言模型，擅长中英文对话、推理和文本生成            |
| **厂商**       | 深度求索 (DeepSeek)                                               |
| **是否免费**   | ❌ 付费，按 Token 计费（输入 ¥1/M tokens，输出 ¥2/M tokens）      |
| **Token 消耗** | 每次 RAG 问答约 4000 tokens（含检索上下文注入）                   |
| **核心功能**   | RAG 问答的生成模型，接收检索到的文档片段 + 用户问题，生成最终答案 |
| **应用场景**   | 知识库问答（RAG 模式）—— 基于文档内容回答用户提问                 |

**配置项：**

- `DEEPSEEK_API_KEY` — API 密钥
- `DEEPSEEK_BASE_URL` — `https://api.deepseek.com/v1`
- `DEEPSEEK_MODEL` — `deepseek-chat`

---

## 2. GLM-4V-Flash (glm-4v-flash)

| 项目           | 内容                                                                 |
| -------------- | -------------------------------------------------------------------- |
| **介绍**       | 智谱 AI 免费多模态模型，支持文本 + 图片理解                          |
| **厂商**       | 智谱 AI (Zhipu AI / BigModel)                                        |
| **是否免费**   | ✅ 完全免费                                                          |
| **Token 消耗** | 纯文本对话约 500-1500 tokens/次；带图片时视图片大小额外消耗          |
| **核心功能**   | 多模态对话，支持图片理解与描述                                       |
| **应用场景**   | 普通聊天模式 —— 不走 RAG 检索，直接与 LLM 对话，支持用户上传图片提问 |

**配置项：**

- `CHAT_API_KEY` — 智谱 API 密钥
- `CHAT_BASE_URL` — `https://open.bigmodel.cn/api/paas/v4`
- `CHAT_MODEL` — `glm-4v-flash`

---

## 3. BAAI/bge-small-zh-v1.5（本地嵌入模型）

| 项目           | 内容                                                       |
| -------------- | ---------------------------------------------------------- |
| **介绍**       | 北京智源研究院 (BAAI) 开源的中文文本嵌入模型，轻量高效     |
| **厂商**       | BAAI (北京智源人工智能研究院) / HuggingFace 开源           |
| **是否免费**   | ✅ 完全免费，本地运行，无需 API Key                        |
| **Token 消耗** | 无 API 调用，零费用；本地 CPU 推理                         |
| **核心功能**   | 将文本转换为 512 维向量，用于语义相似度计算                |
| **应用场景**   | 文档分块向量化入库、用户问题向量化检索、语义缓存相似度判断 |

**配置项：**

- `EMBEDDING_PROVIDER` — 设为 `huggingface` 启用
- 模型自动从 HuggingFace 镜像 (hf-mirror.com) 下载

---

## 4. DashScope text-embedding-v3（备选嵌入模型）

| 项目           | 内容                                                            |
| -------------- | --------------------------------------------------------------- |
| **介绍**       | 阿里云通义千问文本嵌入模型，云端 API 调用                       |
| **厂商**       | 阿里云 (DashScope / 通义)                                       |
| **是否免费**   | ⚠️ 有免费额度，超出后付费（约 ¥0.7/百万 tokens）                |
| **Token 消耗** | 每次嵌入调用按文本长度计费                                      |
| **核心功能**   | 文本向量化，支持更长文本和更高维度                              |
| **应用场景**   | 与 bge-small-zh 相同，作为云端替代方案（适合无 GPU 服务器部署） |

**配置项：**

- `EMBEDDING_PROVIDER` — 设为 `dashscope` 启用
- `DASHSCOPE_API_KEY` — 阿里云 API 密钥

---

## 模型选型总结

```
┌─────────────────────────────────────────────────────┐
│                    用户提问                           │
├──────────────────┬──────────────────────────────────┤
│  普通聊天模式     │       RAG 知识库问答模式          │
│                  │                                  │
│  GLM-4V-Flash    │  ① bge-small-zh 向量化问题        │
│  (免费多模态)     │  ② FAISS 向量检索相关文档片段      │
│                  │  ③ DeepSeek Chat 生成答案         │
└──────────────────┴──────────────────────────────────┘
```

| 场景              | 模型                        | 费用          |
| ----------------- | --------------------------- | ------------- |
| RAG 问答生成      | DeepSeek Chat               | 💰 付费       |
| 普通聊天/图片理解 | GLM-4V-Flash                | 🆓 免费       |
| 文本嵌入（默认）  | bge-small-zh-v1.5           | 🆓 免费本地   |
| 文本嵌入（备选）  | DashScope text-embedding-v3 | ⚠️ 有免费额度 |
